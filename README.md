Tech Stack & Key Concepts

SwiftUI – declarative UI, List, HStack, VStack, AsyncImage.

Combine/ObservableObject – @Published to update UI.

Async/Await – modern concurrency for API calls.

Data Models – Product conforms to Identifiable & Codable.

State Management – @StateObject in view, @State for button alerts.

User Flow

App launches and fetches products asynchronously.

User sees a scrollable list of products with images, title, price, and rating.

User can tap Buy to see an alert without affecting row selection.
